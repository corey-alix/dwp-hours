PTO Calendar Component Test

How to Test Interactive PTO Request Creation:

  1. Make calendar editable: Click "Toggle Readonly Mode" to enable editing
  2. Select PTO type: Click on a legend item below the calendar (PTO, Sick, etc.)
  3. Paint cells: Click on weekday cells (Mon-Fri) to apply the selected PTO type
  4. Edit hours: Click on the hours number in painted cells to change the hours
  5. Submit request: Click "Submit PTO Request" to submit your selections

Note: Weekend cells (Sat/Sun) cannot be selected. Existing PTO entries are shown in color.
